## üìå About The Project

This project is a desktop-based application for securing digital images using a hybrid chaos-based encryption algorithm. It combines **Arnold Cat Map (ACM)** for pixel permutation and **Duffing Map** for diffusion (keystream generation).

The application is designed to protect digital images from unauthorized access and statistical attacks by transforming the original image into a random like cipher image.

**Key Capabilities:**
* **Permutation:** Scrambles pixel positions using Arnold Cat Map to break spatial correlation.
* **Diffusion:** Modifies pixel values using a chaotic keystream generated by the Duffing Map equation.
* **Statistical Analysis:** Includes built-in tools to analyze Histogram, Entropy, and Pixel Correlation of the encrypted images.
